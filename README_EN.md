## ✨ Product Philosophy

**Yiyin (Yiyin Music)** focuses on a local music playback experience, rejecting algorithmic recommendations and information interference.  
Here, music belongs to you.

- 🎧 Local music first, no online dependency  
- 🔒 Fully offline, privacy is secure  
- 🚫 No ads, no recommendations, no comment system  
- 🧘 Immersive listening experience  

---

## 🧱 Technology Stack

- Wails (Go + Native WebView)  
- Vue 3 + Composition API  
- TypeScript  
- Vite  

---

## ⚡ Performance Advantages

- 📦 Lightweight, small package  
- 🚀 Fast startup, smooth performance  
- 🧠 Low memory usage

## 🚀 Core Features

- 📂 Local music scanning  
- 🏷 Tag parsing (ID3 / FLAC / APE)  
- 🎤 Immersive lyrics (LRC & embedded)  
- 🎼 Artist / Album view  
- ⭐ Favorites and playlists  
- 📱 Local network remote control  
- 🌗 Light / Dark theme toggle

## 🎵 Lyrics Explanation

- Prioritize reading tags, no need for `.lrc` files
