## ✨ Philosophy

**Yiyin Music** focuses on local playback, rejecting algorithms and distractions.  

- 🎧 Local-first, no online dependency  
- 🔒 Fully offline, privacy-first  
- 🚫 No ads, recommendations, or comments  
- 🧘 Immersive listening experience  

---

## 🧱 Tech Stack

- Wails (Go + Native WebView)  
- Vue 3 + Composition API  
- TypeScript  
- Vite  

---

## ⚡ Performance

- 📦 Lightweight, small package  
- 🚀 Fast startup, smooth playback  
- 🧠 Low memory usage  
- 🖥 Native-like experience

## 🚀 Features

- 📂 Local music scanning  
- 🏷 Metadata parsing (ID3 / FLAC / APE)  
- 🎤 Immersive lyrics (LRC & embedded)  
- 🎼 Artist / Album views  
- ⭐ Favorites & playlists  
- 📱 LAN remote control  
- 🌗 Light / Dark themes  

---

## 📱 Remote Control

1. Ensure devices are on the **same LAN**  
2. Open "Remote Control" in app  
3. Scan QR code with phone
