## ✨ 产品理念

**依音（Yiyin Music）** 专注于本地音乐播放体验，拒绝算法推荐与信息干扰。  
在这里，音乐只属于你自己。

- 🎧 专注本地音乐，无在线服务依赖  
- 🔒 完全离线运行，保护隐私  
- 🚫 无广告、无推荐、无评论系统  
- 🧘 沉浸式播放体验，回归音乐本质  

---

## ⚡ 性能优势

基于 **Wails** 构建，带来更轻量的桌面体验：

- 📦 **安装包体积小**：整体更精简，下载更轻松  
- 🚀 **运行更流畅**：启动迅速，响应自然  
- 🧠 **内存占用低**：长时间运行依然轻盈稳定  
- 🖥 **原生级体验**：结合系统 WebView，性能与体验兼顾

## 🚀 核心功能

- 📂 **本地音乐扫描**：快速导入并构建你的私人音乐库  
- 🏷 **标签解析**：完整读取 ID3 / FLAC / APE 元数据  
- 🎤 **沉浸式歌词**：支持lrc歌词与标签内嵌歌词  
- 🎼 **歌手 / 专辑视图**：自动分类，清晰管理音乐  
- ⭐ **收藏与歌单**：自由管理你的音乐偏好  
- 📱 **局域网远程控制**：手机扫码控制播放  
- 🌗 **多主题模式**：明亮 / 暗黑 主题切换
