## ✨ 产品理念

**依音（Yiyin Music）** 专注于本地音乐播放体验，拒绝算法推荐与信息干扰。  
在这里，音乐只属于你自己。

- 🎧 本地音乐优先，无在线依赖  
- 🔒 完全离线，隐私安全  
- 🚫 无广告、无推荐、无评论系统  
- 🧘 沉浸式播放体验  

---

## 🧱 技术栈

- Wails（Go + 原生 WebView）  
- Vue 3 + Composition API  
- TypeScript  
- Vite  

---

## ⚡ 性能优势

- 🚀 启动快速，流畅运行  
- 🧠 内存占用低  
- 🖥 原生级体验

## 🚀 核心功能

- 📂 本地音乐扫描  
- 🏷 标签解析（ID3 / FLAC / APE）  
- 🎤 沉浸式歌词（LRC & 内嵌）  
- 🎼 歌手 / 专辑视图  
- ⭐ 收藏与歌单  
- 📱 局域网远程控制  
- 🌗 明亮 / 暗黑主题切换  

---

## 📱 远程控制指南

1. 保证设备在同一局域网  
2. 打开播放器 → 远程控制  
3. 手机扫码连接  
4. 可控制播放：▶ ⏮ ⏭
