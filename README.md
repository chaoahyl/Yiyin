## ✨ 产品理念

**依音（Yiyin Music）** 专注于本地音乐播放体验，拒绝算法推荐与信息干扰。  
在这里，音乐只属于你自己。

- 🎧 **本地优先**：回归纯粹，无在线资源依赖
- 🔒 **隐私安全**：完全离线运行，不上传任何用户数据
- 🚫 **零干扰**：无广告、无推荐、无评论系统
- 🧘 **沉浸体验**：专为 picky ears 打造的高级 UI

## 🚀 核心功能

- 📂 **全量扫描**：深度索引本地音乐文件夹
- 🏷 **标签解析**：支持 ID3 / FLAC / APE 高性能解析
- 🎤 **沉浸歌词**：类 Apple Music 动态歌词效果（LRC & 内嵌）
- 🎼 **多维视图**：按歌手、专辑、收藏夹灵活分类
- 📱 **远程控制**：同一局域网下，手机扫码即可控制电脑端播放
- 🌗 **主题切换**：完美适配明亮与暗黑模式

---

## 📱 远程控制指南

1. 确保电脑与手机处于同一局域网（Wi-Fi）
2. 打开播放器 → 侧边栏“远程控制”
3. 手机扫码连接，即可在手机上控制：▶ 播放/暂停、⏮ 上一曲、⏭ 下一曲

---

## 🎵 歌词说明

- 推荐使用 **LDDC** 或 **MusicTag** 整理歌词
- 支持**嵌入标签歌词**（优先读取），无需额外的 `.lrc` 文件
